OmniBypass (v1.0.0)
===========

This script is used to detect ad blockers.


Valid on
---------------------
- Google Chrome
- Mozilla Firefox
- Internet Explorer (8+)
- Safari
- Opera

Install via
---------------------
Manual:
```
Download "omniBypass.js" and add it to your website.
```

Run and test
---------------------
Go to root directory and run
```
python -m SimpleHTTPServer
 
```
On the browser go to
```
http://localhost:8000/test.html
 
```
